ito — v2 TUI · Digest + issue view

The v2 surface (SPEC §7, in-process over the core per ADR-0002): a Digest — full-width rows grouped by status — a Board — the same issues as one-line kanban cards, truncated with … — and the full issue view that ⏎ opens.

In the real TUI you'd switch view with the number keys (1 digest, 2 board — the digit shows in the header tab), hide / reveal a Digest section with h (the Board always shows every column), and press / to filter or : for commands — both turn the bottom bar into an inline input (the view stays, esc leaves the field). Here, the bar below (or ← / →) cycles all the screens, including the issue detail and mocks of the filter and command line. Read-only.
